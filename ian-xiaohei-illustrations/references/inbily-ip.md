# INBILY IP

## 词识别点

固定识别词：`INBILY粉卫衣星钥少女`。

每次生成都要围绕这个词锁定角色。它绑定三个最强识别点：

- 粉色 INBILY 连帽卫衣
- 粉色星形挂件
- 棕色短发小呆毛

只要这三点不稳，角色就不合格。

## 角色定义

INBILY粉卫衣星钥少女是本 skill 的固定视觉 IP。她不是品牌海报模特，不是贴纸吉祥物，不是普通可爱动漫角色，而是小黑风正文配图里认真参与系统运转的荒诞工作者。

默认每张图都要出现她。她必须承担画面的核心动作，不能只是站在角落、摆 pose 或露出微笑。

## 外形

- 棕色齐肩短发，发尾微微外翘。
- 额前空气刘海，头顶一撮小呆毛。
- 大而圆的棕色眼睛，可以有高光，但不要过度闪亮。
- 圆脸，轻微粉色腮红，表情温暖但克制。
- 亮粉色 oversized 连帽卫衣，正面有简化但清晰的白色 `INBILY` 字母或可读标识。
- 卫衣可以保留少量小星星装饰，但不要画成复杂品牌海报。
- 白色短裤，少量粉色细节。
- 腰侧有粉色星形小挂件，这是第二识别点。
- 白色中筒袜，粉色细条纹。
- 粉白运动鞋，可以有极简 INBILY 小标识。
- 身体比例偏简化 Q 版，但不要幼儿化；适合 1:4 到 1:5 头身比。

## 小黑风角色气质

保留角色识别，但降低商业插画感：

- 用黑色手绘线稿先建立角色，不要靠大面积精致上色。
- 粉色只保留在卫衣、鞋、星形挂件等必要识别点。
- 表情可以温暖，但要有一点 deadpan、认真、正在工作的气质。
- 动作要服务隐喻：搬、拉、推、剪、称、守门、修补、记录、塞进机器、扶梯子、接住路径。
- 角色应像白纸草图中的系统操作员，而不是品牌宣传页的主角。

## 常见职责

让 INBILY IP 承担核心动作：

- 搬运素材、观点、问题或信任盒子。
- 拉线汇聚信息源。
- 卡在断点里修补流程。
- 在怪机器里操作“判断”杆。
- 站在筛选漏斗旁分拣。
- 切开“素材鱼”或把纸团压成多种形状。
- 盖章承接话术。
- 牵着承接路径。
- 举警告牌看坑。
- 从洞里伸手接住内容。
- 在旁边搬砖、搭桥、开门、分拣、记录。

## 禁止

- 不要生成黑色小怪物或黑色吉祥物。
- 不要把她画成真实人物、3D、厚涂、商业海报、精致动漫立绘。
- 不要复制参考图里的粉色边框、标题栏、展示页版式和装饰角标。
- 不要让整张图变成粉色品牌海报；小黑风仍然以黑白线稿和白底为主。
- 不要过度可爱、幼儿化、表情包化、卖萌化。
- 不要性感化、成熟御姐化、复杂服装化。
- 不要让 INBILY logo 乱码成一堆假字；如果无法保证文字，宁可简化成清晰白色短标识。
- 不要让她只站在旁边看图表。
- 不要让她抢走结构表达，角色必须服务核心隐喻。

## 判断标准

如果去掉 INBILY IP，图的核心隐喻还能完全成立，说明她太装饰了；要重写提示词，让她成为动作主体。

如果第一眼只看到“粉色可爱品牌角色”，看不到白底怪诞草图结构，说明不够小黑风；要减少装饰、减少粉色面积、增加黑色手绘线和荒诞动作。
